How it works

One analysis pass, several ways to reclaim space.

Xray My Mac shares scan data across duplicate discovery, storage visualization, uninstall cleanup, and targeted reclaim flows so the app does not keep repeating the same disk walk.

  1. Scan once and build the file inventory.
  2. Use embeddings and reranking to prioritize the best duplicate and cleanup targets.
  3. Inspect the storage map and uninstall residue before acting.
  4. Clear caches, duplicates, and leftovers through a shared safety pipeline.
